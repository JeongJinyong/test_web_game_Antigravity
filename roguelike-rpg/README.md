# Roguelike ARPG

A Diablo-style Roguelike ARPG built with Phaser 3, TypeScript, and Vite.

## Tech Stack

- **Phaser 3**: Game framework (v3.80+)
- **TypeScript**: Language (Strict mode enabled)
- **Vite**: Build tool and dev server

## Project Structure

- `src/`: Source code
  - `scenes/`: Game scenes (Boot, Game, UI)
  - `entities/`: Game entities (Player, Enemy)
  - `systems/`: Game systems (Dungeon, Combat)
  - `config/`: Game configuration
- `assets/`: Game assets (Sprites, Audio)

## Getting Started

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

## Assets

- **Kenney Roguelike/RPG Pack**: Used for tiles and characters.
